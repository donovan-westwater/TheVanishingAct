# TheVanishingAct
This is my foundations of game production project.

Controls: 
Movement WASD (PC),Left stick (gamepad)
Rewind shot Spell -J
Rewind time -L
Teleport Grab -F
Spatial Tunnel -Space
Restart level -B
Save in general slot -M
Load general slot -N
Save in slot 1 -num1
Save in slot 2 -num2
Save in slot 3 -num3
Load save 1 -SHIFT + 1
Load save 2 -SHIFT + 2
Load save 3 -SHIFT + 3


Objective: Steal all the green orbs without getting caught.
